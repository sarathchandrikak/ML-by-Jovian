# ML-by-Jovian

Implementation of Supervised and Un-Supervised Algos using scikit learn along with exploratory data analysis

Supervised Algos covered are:
  1. Linear Regression
  2. Logistic Regression
  3. Decision Tree
  4. Random Forest
  5. XG Boost
  
Unsupervised Algos covered are:
  1. KNN
  2. Kmeans clustering
